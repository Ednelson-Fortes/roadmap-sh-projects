# 📄 Personal Portifolio - Roadmap.sh Project

Este projeto é a implementação do desafio de [Personal Portifolio](https://roadmap.sh/projects/portfolio-website) do Roadmap.sh. O objetivo foi desenvolver um currículo digital de alta performance, priorizando o uso de HTML5 semântico para SEO e acessibilidade. O projeto foca na organização modular do CSS e na estruturação de dados de forma que a experiência do usuário (UX) seja intuitiva e otimizada para diferentes dispositivos

## 🚀 Como Visualizar
1. Clone o repositório: `git clone https://github.com/Ednelson-Fortes/roadmap-sh-projects.git`
2. Navegue até a pasta: `cd Single-Page_CV`
3. Abra o arquivo `index.html` em seu navegador de preferência.
