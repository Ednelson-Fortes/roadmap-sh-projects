# 📄 Single-page CV - Roadmap.sh Project

Este projeto é a implementação do desafio de [Single-page CV](https://roadmap.sh/projects/single-page-cv) do Roadmap.sh. O objetivo foi construir um currículo profissional utilizando as melhores práticas de estrutura e visibilidade na web.

## 🚀 Como Visualizar
1. Clone o repositório: `git clone https://github.com/Ednelson-Fortes/roadmap-sh-projects.git`
2. Navegue até a pasta: `cd Single-Page_CV`
3. Abra o arquivo `index.html` em seu navegador de preferência.
