# 📄 Image Grid Layout - Roadmap.sh Project

Este projeto é a implementação do desafio de [Image Grid](https://roadmap.sh/projects/image-grid) do Roadmap.sh. O objetivo central foi dominar o sistema de coordenadas bidimensional do CSS Grid, criando uma galeria de imagens que se organiza de forma automática e inteligente, independentemente da quantidade ou proporção das imagens inseridas.

## 🚀 Como visualizar 

1. Clone o repositório: `git clone https://github.com/Ednelson-Fortes/roadmap-sh-projects.git`
2. Navegue até a pasta: `cd Image-grid-layout`
3. Abra o arquivo `index.html` em seu navegador de preferência.
