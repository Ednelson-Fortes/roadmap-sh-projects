# 📄 Basic HTML Website - Roadmap.sh Project

Este projeto é a implementação do desafio de [Basic HTML Website](https://roadmap.sh/projects/basic-html-website) do Roadmap.sh. O objetivo deste projeto é ensinar como estruturar um site usando HTML, ou seja, diferentes seções de um site, como cabeçalho, rodapé, navegação, conteúdo principal, barras laterais etc

## 🚀 Como Visualizar
1. Clone o repositório: `git clone https://github.com/Ednelson-Fortes/roadmap-sh-projects.git`
2. Navegue até a pasta: `cd Basic-HTML-Website`
3. Abra o arquivo `index.html` em seu navegador de preferência.
